Environmental sound extraction using onomatopoeia
　　

Author: Yuki Okamoto, Shota Horiguchi, Masaaki Yamamoto, Keisuke Imoto, Yohei Kawaguchi


This is a demonstration of environmental sound extraction using onomatopoeia. We conducted environmental sound extraction using one proposed method and two comparison methods as follows:

    　
  • Superclass-conditioned method (Baseline)
    • This method is conditioned by superclass sound event class.
    　
  • Subrclass-conditioned method (Baseline)
    • This method is conditioned by subclass sound event class.

  • Onomatopoeia-conditioned method (Proposed)
    • This method is conditioned by onomatopoeia. 　　

As sounds of the dataset, We selected 16 types of sound events in superclass and 44 types of sound events in subclass from RWCP-SSD (Real World Computing Partnership-sound Scene Database) [1]. For the onomatopoeia corresponding to each sound sample, we used the dataset in RWCP-SSD-Onomatopoeia [2].
